# Flexpert

This repository contains a React + TypeScript + Vite shopping cart billing application.

## Project summary
- A responsive shopping cart experience with real-time bill calculations.
- State is managed with Redux Toolkit.
- The UI uses Tailwind CSS.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Main folders
- src/components: UI components
- src/store: Redux store and basket slice
- src/data: product and offer data
